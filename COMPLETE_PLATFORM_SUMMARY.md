- AI photo/video verification (deepfake detection)
- Company background checks
- Asset monitoring & management

- Mobile money payments (PawaPay)

## 🔌 **API ENDPOINTS** (14 Total)

### **Public Security Checks** (3)
- POST `/api/public-check/url` - Website security scan
- POST `/api/public-check/app` - Mobile app check
- POST `/api/public-check/company` - Company verification

### **AI Verification** (4)
- POST `/api/ai-verify/photo` - Photo authenticity check
- POST `/api/ai-verify/video` - Video deepfake detection
- POST `/api/ai-verify/photo/upload` - Upload photo file
- POST `/api/ai-verify/video/upload` - Upload video file

### **Payments & Subscriptions** (5)
